Classes

- :py:class:`AbstractModel`:
  Abstract machine learning model which exposes input and output names, run and save methods.

- :py:class:`Ensemble`:
  Ensemble model facilitates assembling multiple models into one for more accurate predictions.

- :py:class:`Sequence`:
  Model sequence provides simple abstraction for sequential application of multiple models to input batches.
